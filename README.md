# VerificadorMD2
Verificação da integridade de multiplos arquivos com o algoritmo MD2

Uso: python script.py HASH PASTA  

Onde:
HASH - um arquivo .txt contendo os valores de hash separados por quebra de linha  

PASTA - pasta/diretório contendo os arquivos a serem verificados a integridade, também em .txt (por enquanto)
